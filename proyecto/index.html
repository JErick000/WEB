¿Porque deberias de confiar con nosotros?

En Divbast nos enfocamos en brindar un servicio adecuado a nuestros cliente, desde marketing hasta nuestros propios paquetes de servicios web, en Divbast buscamos apoyar al medio ambiente mediente concientisacion en nuestras campañas de marketing hasta juntar fondos para el cuidado del medio ambiente, nuestros principales metodos se basan en trabajos remotos, evitando que nuestros empleados se movilizen en transportes que puedan resultar contaminamtes para el medio ambiente, nuestra empresa además es completamente virtual para que nuestros trabajadores puedan dar su mejor rendimiento desde la comodidad de sus casas por ultimo nuestros principales paquetes llevan los nombres de las mayores especies en peligro de extinción desde el paquete mas comodo en el cual casi todos pueden acceder hasta el paquete mas caro que no todos pueden acceder, mediante este metodo buscamos mostrar las especies con menor y mayor estadistica de peligro en el año actual.

Paquetes

- BIRD: Ideal para pequeñas empresas que de momento solo buscan brindar informacion. Este paquete ofrece paginas con la misma plantilla y poco diseño (El soporte, modificaciones, servidor y host se ofrecen por serparado, el precio varia de acuerdo a las páginas)

- MAMMALIAN: Ideal para empresas medianas/pequeñas, Con este paquete obtienes un mejor producto, un proyecto con distint estructura en las distintas paginas y un mejor diseño (El soporte, modificaciones, servidor y host se ofrecen por serparado, el precio varia de acuerdo a las páginas)

- CORAL REEF: recomendado para empresas medianas, Con este paquete podras obtener un mejor diseño por nuestros trabajadores especialisados en css y boostrap, ademas de una mejor funcionalidad de dicha página, teniendo la capacidad de agregar otros lenguajes como jscript(para diseños) ademas de poder colocar enlaces y un soporte para cualquier duda que tengas, (modificaciones, servidor y host se ofrecen por serparado, el precio varia de acuerdo a las páginas)

- SHARK: Ideal para empresas medianas/grandes, con este paquete obtendras un mayor disño y funcionalidad de tu pagina, podras tener base de datos, jscript, php ádemas de un servicio de soporte y 5 modificaciones al mes (servidor y host se ofrecen por serparado, el precio varia de acuerdo a las páginas)

- CÍCADAS: este paquete es distinto a todos los antes mencionado, este paquete es un paquete personalizado en el cual puedes solicitar desde el paquete mas barato pero con una base de datos o desde el paquete con mayor diseño pero con un contenido simple, etc todo es a eleccion del cliente ademas de tener soporte y 10 modificaciones al mes (puede aumentarse) este paquete tiene un costo mayor pero tendrás la pagina ideal que tienes en mente (servidor y host se ofrecen por serparado, el precio varia de acuerdo a las páginas)
